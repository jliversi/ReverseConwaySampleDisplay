Reversing Conway's Game of Life

Early in my programming/computer science journey, I became fascinated with Conway's Game of Life. One problem I was particularly interested in solving was whether you could play the game "backwards". That is, given a state X, can you produce the previous state Y?

Using python, I built a small project which takes 2D patterns and outputs a previous state. This site serves as a small sample of some of those patterns (mostly text as those are the easiest "meaningful" patterns to fit into such a small resolution). For the full source code and write up on the program which produces these previous states, check out the github repo here

To use this site, load a pattern using the dropdown. Then either click to run the Game of Life, or click to go step by step. After three steps, the intended pattern will be produced.
